## The Setup

- **Brain:** Raspberry Pi 5 (8GB) running Raspberry Pi OS
- **AI:** Claude (via OpenClaw gateway)
- **Voice:** ElevenLabs TTS through a USB speaker
- **Eyes:** Logitech C920 webcam
- **Face:** ILI9341 2.8" TFT display showing a pixel art Dobby sprite
- **Ears:** Push-to-talk button (GPIO24) + M5Stack Atom Echo
- **Messaging:** Telegram (primary), Bluesky (@dobby1738.bsky.social)
